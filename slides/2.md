
# Operadores matemáticos

``` c++
// Para SUMAR, utilizaremos {variable1} + {variable2}, por ejemplo 3 + 2 o "Francisco" + "Galvez"
int mySum = 32 + 54;

// Para DIVIDIR, utilizaremos {variable1} / {variable2}, por ejemplo, 10 / 2
float myDivision =  10 / 2;

// Para obtener el MÓDULO de una división, utilizaremos {variable1} % {variable2}, por ejemplo 10 % 2
int myModule = 10 % 2;

// Para MULTIPLICAR, utilizaremos {variable1} * {variable2}, por ejemplo 3 * 2

int myMultiplication  = 3 * 6;

// Para COMPARAR, utilizaremos {variable1} == {variable2}, por ejemplo 3 == 2 --> Esto retorna un booleano. True o false
bool areEqual = "Coquita" == "Pepsi"
```
